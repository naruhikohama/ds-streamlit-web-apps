Quais as características dos usuários winback?¶

A Acerto uma empresa de Winback, ou seja, somos especialistas em reconquistar clientes desengajados e recuperar receitas perdidas. Para isso, podemos expor consumidores a estratgias diferentes esperando sua converso.

O case abaixo est neste contexto, em que escolhemos os melhores consumidores a serem acionados, com base no resultado do acionamento de outros consumidores.

Objetivo¶

Crie um script com um modelo preditivo simples utilizando esses dados.

Passos¶

  • Separe a base em treino e teste (25% teste).
  • Desenvolva o modelo.
  • Retorne a acurcia de seu modelo.
  • Gere uma lista com os 1.000 melhores consumidores a serem acionados.

Observao: use o pagamento como varivel dependente.

Outras informações¶

Em conversa, foi exposto como a forma de abordar o cliente de maneira mais especfica ajuda na converso do cliente. Dessa forma, um modelo puramente preditivo, embora provavelmente apresente uma acurcia maior, no necessariamente ajuda na abordagem do cliente para sua converso.

Uma metodologia puramente preditiva pode ajudar a entender uma carteira e seu potencial de winback. No entanto, a partir do momento que o objetivo fazer a converso, importante entender quais so os fatores que fazem aquele indivduo ter um alto potencial de converso.

Aplicação de algoritmo¶

Para o modelo final, teremos muito menos observaes para realizar as comparaes. No entanto, um modelo com todos os dados traria uma acuracidade ilusria. Quando a maior parte dos dados de uma categoria em particular, o modelo pode se tornar "preguioso" e fazer a previso que todos pertencem ao maior grupo. Isso manter a acuracidade alta, mas no ajudar no problema de negcio principal, que identificar quem so aqueles com maior probabilidade de gerar converso.

Com a base balanceada e sabendo dos problemas que uma base no balanceada poderia trazer, vamos seguir para a separao da base em treino e teste.

Em seguida a aplicao do modelo catboost, que semelhante a um modelo de random forest.
