Quais as características dos usuários winback?¶

A Acerto é uma empresa de Winback, ou seja, somos especialistas em reconquistar clientes desengajados e recuperar receitas perdidas. Para isso, podemos expor consumidores a estratégias diferentes esperando sua conversão.

O case abaixo está neste contexto, em que escolhemos os melhores consumidores a serem acionados, com base no resultado do acionamento de outros consumidores.

Objetivo¶

Crie um script com um modelo preditivo simples utilizando esses dados.

Passos¶

  • Separe a base em treino e teste (25% teste).
  • Desenvolva o modelo.
  • Retorne a acurácia de seu modelo.
  • Gere uma lista com os 1.000 melhores consumidores a serem acionados.

Observação: use o pagamento como variável dependente.

Outras informações¶

Em conversa, foi exposto como a forma de abordar o cliente de maneira mais específica ajuda na conversão do cliente. Dessa forma, um modelo puramente preditivo, embora provavelmente apresente uma acurácia maior, não necessariamente ajuda na abordagem do cliente para sua conversão.

Uma metodologia puramente preditiva pode ajudar a entender uma carteira e seu potencial de winback. No entanto, a partir do momento que o objetivo é fazer a conversão, é importante entender quais são os fatores que fazem aquele indivíduo ter um alto potencial de conversão.

Aplicação de algoritmo¶

Para o modelo final, teremos muito menos observações para realizar as comparações. No entanto, um modelo com todos os dados traria uma acuracidade ilusória. Quando a maior parte dos dados é de uma categoria em particular, o modelo pode se tornar "preguiçoso" e fazer a previsão que todos pertencem ao maior grupo. Isso manterá a acuracidade alta, mas não ajudará no problema de negócio principal, que é identificar quem são aqueles com maior probabilidade de gerar conversão.

Com a base balanceada e sabendo dos problemas que uma base não balanceada poderia trazer, vamos seguir para a separação da base em treino e teste.

Em seguida a aplicação do modelo catboost, que é semelhante a um modelo de random forest.
